## Configuration

**Environment:**
- Supabase URL and keys via `.env.local`:
  - `NEXT_PUBLIC_SUPABASE_URL` - Supabase project endpoint
  - `NEXT_PUBLIC_SUPABASE_ANON_KEY` - Public client key for frontend
  - `SUPABASE_SERVICE_ROLE_KEY` - Admin key (server-side only, never exposed)
  - `NEXT_PUBLIC_APP_NAME` - Application display name
  - `NEXT_PUBLIC_APP_URL` - Application base URL
- Template: `.env.local.example` documents all required variables

**Build:**
- `next.config.mjs` - Next.js configuration (React strict mode, image optimization)
- `tsconfig.json` - TypeScript strict mode, path aliases:
  - `@/*` → `./`
  - `@/components/*` → `./components/*`
  - `@/lib/*` → `./lib/*`
  - `@/types/*` → `./types/*`
  - `@/app/*` → `./app/*`
- `tailwind.config.ts` - Tailwind design tokens (colors, typography, animations)
- `postcss.config.mjs` - PostCSS plugins (Tailwind, Autoprefixer)
- `.eslintrc.json` - ESLint rules (extends Next.js and Prettier)
- `.prettierrc` - Code formatting (100 char width, tabs=2, Tailwind plugin)
